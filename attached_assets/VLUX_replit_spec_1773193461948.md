## 🚫 現時点でデモに含めないもの（本番実装まで保留）

| 項目 | 理由 |
|------|------|
| 実際のClaude API呼び出し | Replitにキーを置かない。デモはモック固定データで十分 |
| Apple HealthKit実連携 | iOS実機が必要。デモはモックデータで演出 |
| Supabase実DB | セキュリティリスク。デモはReactのstateのみで動作 |
| Stripe実決済 | 不要。金額表示のみで十分 |
| リアル患者データ | 個人情報保護。田中大輔（仮）の固定データのみ使用 |
